# BUFIA-AR

**BUFIA-AR** is a Python tool for tone and syllable structure analysis using Autorep representations and a grammar induction algorithm (BUFIA).

---

## Features

- Convert syllabified wordlists into Autorep objects.
- Induce tonal grammar rules based on user-defined complexity thresholds.
- Visualize tone-syllable-mora associations with Graphviz diagrams.
- Supports multiple output formats (SVG, PNG, PDF, etc.)

---

## Requirements

- Python 3.7+
- Packages:
  - pandas
  - numpy
  - graphviz
  - argparse
